# Шаблон описания перевода для Nexus (BBCode)

Формат — Nexus BBCode. Используется для модов-переводов (русификаторов) других
модов. Поддерживает мульти-мод (2-в-1 и т.п.). Блоки идут строго в этом порядке.

## Структура (порядок блоков)
1. **Шапка** (`[center]`): `[size=5][b]{{NAME}}[/b]` + подзаголовок
   `[size=3][color=#D4D4D8]Русский перевод для {{ORIGINALS}} · {{GAME}}[/color]`
2. **📖 Описание** — что переведено; ссылки `[url=...]` на оригиналы; оговорки
   (напр. «для Better MSC переведены ТОЛЬКО НАСТРОЙКИ МОДА»); «оригинальные
   файлы не изменяются».
3. **🌍 Язык** — Русский.
4. **📦 Требования** — `[list]` оригинальные моды (со ссылками) + `MSCLoc API`.
5. **🔧 Установка** — `[list=1]`: установить оригиналы → скопировать папку Mods
   в корень игры → запустить игру.
6. **🛠️ Чем сделан перевод** — блок про **ULTIMA** + ссылка
   `https://www.nexusmods.com/mysummercar/mods/11554` (MSC-страница ULTIMA;
   BG3 — 7410002).
7. **🙏 Благодарности** — авторы оригинальных модов.
8. **Подвал** (`[center][size=2][color=#71717A]`): «Перевод: ANICKON · Сделано в ULTIMA».

## Заголовок/именование
- Один мод: `[Оригинал] — Russian Translation (RU)` / `[Оригинал] RU`.
- Набор: `[Мод A] + [Мод B] — Russian Translation (N-in-1)` или
  `Русификатор: A + B`.

## Стиль/правила
- Заголовки блоков: `[size=4][b]…[/b][/size]`; текст: `[size=3][color=#D4D4D8]…[/color][/size]`.
- На Nexus НЕТ `[hr]`. Категория для перевода — Translations (или Miscellaneous),
  не «Saved Games».
- Краткое описание ≤ 350 символов; можно двуязычно (🇷🇺 сначала, 🇬🇧 потом).
