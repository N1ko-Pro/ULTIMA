# Serena Memory — правила использования

У меня есть доступ к инструментам памяти Serena (MCP-сервер `serena`).
Память — это markdown-заметки о проекте, которые хранятся в `.serena/memories/`
и переживают перезапуски сессии. Я должен использовать их проактивно по
ключевым словам пользователя, не дожидаясь явной команды «вызови инструмент».

## Когда ЗАПОМНИТЬ (вызвать `write_memory`)

Триггеры (рус/eng, в любой форме и регистре):
«запомни», «запиши в память», «сохрани в память», «заметка», «не забудь»,
«remember», «save to memory», «note this».

Действия:
1. Определить осмысленное имя памяти (латиницей, можно с «/» для тем,
   напр. `architecture/backend`, `conventions`, `decisions/auth`).
2. Сформировать содержимое в markdown: кратко, структурировано, по делу.
3. Вызвать `write_memory` с этим именем и содержимым.
4. Если по теме уже есть заметка — предпочесть `edit_memory` или дополнить
   существующую, а не плодить дубликаты (сначала свериться с `list_memories`).
5. Кратко подтвердить пользователю, что и под каким именем сохранено.

## Когда ВСПОМНИТЬ (вызвать `read_memory` / `list_memories`)

Триггеры:
«вспомни», «что ты помнишь», «что в памяти», «достань из памяти»,
«recall», «what do you remember», «check memory».

Действия:
1. Вызвать `list_memories`, чтобы увидеть доступные заметки.
2. Прочитать релевантные через `read_memory` (одну или несколько).
3. Если уточнения нет — сначала показать список имён, затем прочитать
   наиболее подходящие по теме запроса.

## Другие операции с памятью

- «обнови память / поправь заметку» → `edit_memory` (или `write_memory` поверх).
- «удали из памяти / забудь» → `delete_memory` (удаление подтвердить, т.к. необратимо).
- «переименуй заметку» → `rename_memory`.

## Принципы

- Имена памяти — латиницей, содержимое — можно на русском.
- В начале содержательной работы над незнакомой областью кода стоит
  свериться со списком памяти (`list_memories`), чтобы переиспользовать контекст.
- Не сохранять в память секреты (токены, пароли, ключи).
- Память коммитится в git вместе с проектом — учитывать это при записи.
